Example :: Factorial 


          def leq   { l r t f -> native "leq"   } // l=<r?t:f
          def mult  { l r     -> native "mult"  }
          def minus { l r     -> native "minus" }

          def cond { c t f -> c t f () }

          def fact { a ->
              cond (leq a 1)
                   { 1 }
                   { mult a (fact (minus a 1)) }
          }

From concrete syntax to abstract syntax 


 [_] :: Sequence Token → Ast

 [i]          = ident(i)                    if i ∈ IDENT
 [n]          = constant(n)                 if n ∈ NUMBER
 [s]          = constant(n)                 if s ∈ STRING
 [{ a -> b }] = abstraction(a, [b]))        -- currified
 [a b]        = application([a],[b])
 [native s]   = native(n)
 [( a )]      = [a]
 [$ a]        = [a]

Transform

Ast → AstDeBruijn

De Bruijn index

  • Invariant with respect to α-conversion
  • Direct access in an environment e.g. registers

Determine De Bruijn indexes 


 [_] :: Ast → [String] → AstDeBruijn

 [ident(n)]e          = variable(i)     if ∃ i / e[i]=n
 [ident(n)]e          = ident(n)        if ∀ i / e[i]≠n
 [abstraction(a,b)]e  = abstraction([b](a::e))
 [application(a,b)]e  = application([a]e, [b]e)
 [constant(c)]e       = constant(c)
 [native(c)]e         = native(c)

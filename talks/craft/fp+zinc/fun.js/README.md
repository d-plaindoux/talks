## Grammar

```
s0 ::=
   definition*

definition ::=
   "def" IDENT Exp
   "do" Exp

Exp ::= 
   SExp+    	       	   

SExp ::= 
   IDENT+ "->" SExp 	         
   "(" Exp ")"
   "(" ")"
   "$ Exp
   NUMBER 
   STRING
   "native" STRING NUMBER
```

## Example

```
def cond native "equal" 3
def equal native "equal" 2
def mult native "mult" 2
def minus native "minus" 2

def fact a -> (cond (equal a 0)
                    (_ -> 1)
                    (_ -> mult a $ fact $ minus a 1) ())

do fact 6
```
